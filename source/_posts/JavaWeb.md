### HTML CSS

* 控制页面结构
* 超文本标记语言

#### CSS

* 控制页面样式
* 层叠样式表
  
#### 基础标签和样式

``<img>``图片标签

* src 指定url
* url-->绝对路径(绝对磁盘/绝对网络) 
        相对路径(./当前目录  ../上一目录)
* width 像素/百分比
* height
  
``<h1>-<h6>``标题标签

* 标题样式 引入CSS
  
``<hr>``水平线标签
